「Pecha cucha link up project 1 Guide for Artist」では、韓国のアート、舞踊分野で活動する若手アーティストたちが一日限りのゲリラ展示を行います。
また、同日18時からは「Pecha cucha link up project 1 Guide for Artist」と題し、それぞれのアーティストが自身の作品や活動について発表します。
「Pecha cucha(ペチャクチャ)」は、2003年にイギリスの建築家二人が日本で始めたプレゼン方式で、韓国でも最近では頻繁に用いられています。
自らが発表しようとする内容を一枚のスライドに20秒、全部で20枚以内のスライドを使って計400秒以内にどれだけインパクトを与えられるのかが鍵となります。
この他にもアーティストによるフードパフォーマンスや公演など、盛りだくさんの内容です。
韓国の若手アーティストが生み出す一日限りのライブ感をぜひお楽しみください！
