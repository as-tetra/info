WACO / YUE TAKEUCHI / KUMIO KURACHI

濃い味

date : Nov. 12, 2017
time : 19:00 open / 19:30 start
ticket : 1500yen + 1drink order (500yen)

出演：WACO / 竹内ゆえ / 倉地久美夫

日時：2017年11月12日
開場：19:00 / 開演：19:30
料金：1500円+1ドリンク（500円）

比較的、出演者平均年齢の高いテトラのイベントの中でも、さらに一段平均年齢の高いイベント。

人生の酸いも甘いも、辛いも苦いも噛み分けた、渋い大人三人による、芳醇なる言葉と音の饗宴、その名も「濃い味」！

これは見逃せません！

WACO（舞/詩/歌・G）

東京での演劇活動を経て、89年よりダンスカンパニー「 ダンスバターTOKIO 」に参加。
伊、独、国内にて多数公演。
その後ソロとなり、即興演奏家と即興舞のセッションライブ多数。
98年より自作のうたの音楽活動開始。
09年、全国的詩の朗読トーナメント「 詩のボクシング 」にて、全国チャンピオン。

現在自作の歌、即興舞、詩の朗読のバラエティソロライブを全国展開中。

http://sanomakiko.web.fc2.com


竹内ゆえ（歌・G）

1980年頃から唄いはじめる。
弾き語りと、自信のバンド「竹内ゆえと猫の倫理」でも活躍。
語りを入れた唄も多く、CD作品として
「ベクトルの逆転した日」「凛々」「美味しい混沌（カオス）の作り方」
「耳の奥」（ラジオドラマ的朗読作品集）「対話」「さかな」「沌沌」がある。
動画作品「カテドラルにて」「×0」等。

「あなたの景色に違う風景を！」と日々活動中

https://www.facebook.com/yue.takeuti


倉地久美夫（歌・G）

64年：福岡県生まれ。
母校中学校舎火事のさい打楽器オルガン等数種を盗み,以後多重録音音楽製作を行う。
関東にて漫画・舞踏～宅録と自身のバンドでライブなど行う。
歌手清水泰之氏、ギタリスト松下隆二氏に師事。
近年は福岡県在住。弾き語りを中心に共演等国内で行う。
11年倉地を追ったドキュメンタリー映画「庭にお願い」公開。
6枚のCDアルバムをリリース。

http://hirunohikari.com/kurachikumio.html
