IDE Matsuri

イデまつり

Nov. 4th, 2017
15:00 start　1500 yen

Yousuke Ide / Gensaku Shirabe / Masanori Matsuda /
Kazuki Fukue / Takeshi Yamamoto / Souta Setoguchi / Tetsuro Matsumoto /
Toshihide Hashimoto / Masayuki Nii / Yuji Nakamura ....and more

開催日：2017年11月4日　15:00 開始
参加費：1500円

出演：井手陽介 / 調原作 / Masanori Matsuda / 福江一起 /
山本武賜 / セトグチ ソウタ / Toshihide Hashimoto /
Tetsuro Matsumoto / Masayuki Nii / duenn / 中村勇治　他

福岡にてこれまで主に実験的な音楽活動をしていた井手陽介氏の結婚記念イベントを開催いたします。

当日は、井手氏とゆかりのある方々に集まってもらい音楽セッションを中心に軽食をつまみながら緩やかに進行する予定です。

基本は、クローズドイベントの形となりますが井手氏へ直接お祝いのメッセージのある方やイベントへのご参加を希望される方はご自由ご参加下さい。
また、当日は参加費として1500円徴収させて頂きます。

※注意点としまして、かなり内輪な内容になりますのでその点ご認識をお願いします。
