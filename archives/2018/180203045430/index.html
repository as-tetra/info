Pecha Cucha Link up projectが帰って来ました!
昨年11月のペチャクチャが韓国アーティストたちで構成された紹介だったとするならば、今回のプロジェクト2は日韓アーティストが一緒に参加して交流します！
今日から2月2日まで1人ずつ参加アーティストの紹介をする予定ですので是非ご覧ください！
それではPecha Cucha Link up project 2の詳細です^ ^

Pecha Cucha Link up projectは韓国と日本でキュレーターとして活動しているキム・ミギョ(Migyo Kim)と小栗栖まり子が共に企画するアーティスト国際交流プロジェクトです。過程と瞬間を重視するこの交流プロジェクトでそれぞれ企画者/芸術家たちはひたすら自分自身と自身の芸術作品だけを表現し提示します。
逐次通訳で進行するペチャクチャスタイルのアーティストトークとたった一日のみ行われるゲリラ展覧会はプロジェクト以降それぞれの自由な交流と制作に対して責任を持ちません。ただ出発点となることを願って応援するのみです。
