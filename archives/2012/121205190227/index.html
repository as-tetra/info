Reading Anthropological theory of Social Values

須崎読書会「第一回マルクスと人類学」Terrence Turner　‘Marxian Value Theory, an anthropological perspective’を読む

2012.12.9(sun)
start: 14:00
￥500(include 1drink)

期日...2012年12月9日(日)
時間...14:00〜
料金...ドリンク代500円
参加希望の方は、bossadelicjp@gmail.com（森）までご連絡ください

資本主義経済の価値とは異なる視座を人類学、マルクス主義から拾うのは、昔から語られていますが、
この際、もう一度、あるいははじめて入門するのに、ターナーの論文がヒントを与えてくれたら、いいな、と思っています。
読書会では、英語文献をド頭から丁寧に読んでいきます。あーでもこーでもない、と喧々諤々の議論をしつつ、価値について、探ってみませんか？

（森元斎/哲学）

価値とは一体何だろう？

価値＝お金とは本当なのだろうか？
人間の労働はすべてお金で計算することができるのだろうか？

そもそも、お金だけが、人間社会における諸活動の価値を測定する唯一の指標であると
考えることに無理がないだろうか？

この研究会を通じて、私たちの生活/現実を規定していると信じられている資本主義経済
の諸概念そのものを批判的に問い直し、人類学のフィールドから人間の行為と価値の多
種多様なあり方を参照していくことで、これまで資本主義社会の中で覆い隠されてきた
人間の諸価値を再び現在の私たちの生活の中に反映させ、新たな社会的生産/再生産の
領域を組織化するための理論的視座を探りたいと考えています。

シリーズ初回は「マルクスと人類学」をテーマに、Terrence Turnerの「Marxian Value
Theory, an anthropological perspective」を読んでいく予定です。

Terence Turnerは、アメリカのマルクス主義人類学者であり、ブラジルの先住民族社会
カヤポ族のフィールドワークとマルクスの価値理論を通じて、国家や市場なき社会におけ
る価値の生産を、人間労働とそれが生み出す成果の表象との関係として、唯物論と
記号論を横断するアプローチで分析しています。

社会的な富とは、誰が生産し、何によって表象され、そして誰が享受しているのか。
マルクスの価値理論を再考することで、この価値を巡る生産関係と、価値の表象•流通•
消費のプロセスを構造的に捉え、表象の下に覆い隠されてしまった、人間の諸行為/労働
へと肉薄し、それらが最終的にどのような社会の全体性を支えているのかという動的な
価値生産のプロセスを見ていくための弁証法的視点を一緒に身につけていきたいと思います。


（江上賢一郎/文化人類学）
