I will make Installation using entire the space,by using image of Whale,Bear,and Japanese archipelago.

日本列島と鯨と熊をイメージしたものを使って、テトラの空間一杯にインスタレーションします。

私には図と地が反転したり、内と外が入れ替わったり、その境界線が形を作ったりすることが面白いので、この三つを選びました。レットウは固有名詞のようなところがあるので気を配らなければと書きます。レットウは弓なりに曲がっていますが、農作業をしていると腰を曲げる作業が多いなと思います。それもそのはず、大地を手で触る作業が多いからです。腰曲げは働く象徴のような気もしますので、この展覧会名で布や絵画を使ったインスタレーションで展示します。

牛島智子

1958福岡県八女市生まれ
1981九州産業大学芸術学部美術学科卒業
1985Ｂゼミ修了

1980年よりグループ展、個展で発表する
1983年より、美術造形教育に携わる
1990年頃よりアートワーク、ＡＩＲ参加する

人生の3分の１を福岡で過ごし、3分の一を神奈川で過ごし、３分の壱をまた福岡で過ごしています。ばらばらにあったものが、体力の限界か、はたまた知恵がついたのか統合されてきています。
